Graph - Number Of Users Per Day:

Line Graph: Display the number of users on the y-axis and dates on the x-axis. Each data point on the graph represents the number of users on a specific day.

Graph - Popular Activities Played:

Bar Chart: Display different activities (scanning, games, trivia, sharing) on the x-axis and the frequency or number of kids participating in each activity on the y-axis. Each bar represents the popularity of a specific activity.

Graph - Most Popular Kids Gender and Age

Bar Chart/Grouped Bar Chart: Display age groups (e.g., 5-8, 9-12, 13-15) on the x-axis and the number of kids falling within each age group on the y-axis. You can represent each gender with a different color or side-by-side bars for easy comparison.

Graph - Channel Distribution (SMS/QR)

Pie Chart: Represent the distribution of channels (SMS, QR code) using different segments of a pie chart. The size of each segment corresponds to the percentage of kids using each channel.
